====== retroarch ======
Created Monday 18 January 2021

	$ sudo pacman -S retroarch
start with
	$ retroarch
to create folders/files in ~/.config
modify retroarch.cfg soas to have filepaths set to home folder for cores/assets
then in settings, enable menu view of online updater, core finder
update cores, update assets
get:
	DS - melonDS
	GBA - mGBA
	SNES - bSNES9x 2005
	
citra and dolphin cores don't work
