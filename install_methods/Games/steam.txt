====== steam ======
Created Saturday 16 January 2021

enable multilib in pacman.conf (uncomment both lines)
	[multilib]
	Include = /etc/pacman.d/mirrorlist
	
then pacman -S steam

in steam settings -> steam play, enable proton usage for more games
