====== Zim ======
Created Thursday 04 February 2021

$ sudo pacman -S zim

Graphical version works as it does on windows
headless, all notes are .txt files
sub pages are made in folders sharing the same name as the page they're below

to run the local server do 
	$ zim --server [notebook name]
	can also add --port [port]
